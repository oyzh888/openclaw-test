# CIFAR-10 深度学习实验

## 概述

本实验在 NVIDIA A100 GPU 上进行 CIFAR-10 图像分类任务，验证深度学习训练流程和 GPU 性能。

## 实验环境

- **GPU:** NVIDIA A100-SXM4-80GB (80GB VRAM)
- **框架:** PyTorch 2.4.1
- **CUDA:** 12.7
- **数据集:** CIFAR-10 (60,000 张 32x32 彩色图片，10 类别)

## 模型架构

```python
SimpleCNN:
  - Conv2d(3, 32, 3) + ReLU + MaxPool
  - Conv2d(32, 64, 3) + ReLU + MaxPool
  - Linear(64*8*8, 128) + ReLU
  - Linear(128, 10)
```

## 训练结果

### 性能指标
- **训练时长:** 9.14 秒 (2 epochs)
- **训练样本:** 50,000 张
- **测试样本:** 10,000 张
- **Batch Size:** 128

### Loss 曲线
```
Epoch 1:
  Batch 100: 1.764
  Batch 200: 1.431
  Batch 300: 1.338

Epoch 2:
  Batch 100: 1.132
  Batch 200: 1.109
  Batch 300: 1.075
```

## 关键发现

### ✅ 优势
1. **机械物体识别优秀** - Truck, Plane, Ship 识别率高
2. **训练速度快** - A100 GPU 性能出色
3. **Loss 下降稳定** - 无过拟合迹象

### ⚠️ 改进空间
1. **动物类别识别较弱** - Cat, Deer, Bird 准确率低
2. **训练轮数不足** - 仅 2 epochs，可提升至 10-20
3. **网络深度有限** - 简单 CNN，可尝试 ResNet

## 改进建议

1. **增加训练轮数** → 目标 10-20 epochs
2. **数据增强** → 翻转、旋转、裁剪
3. **更深网络** → ResNet-18/34, EfficientNet
4. **学习率调度** → 余弦退火或阶梯衰减
5. **正则化** → Dropout, Weight Decay
